Tema 4: Grundlæggende Animation

I Tema 4: Grundlæggende Animation blev vi præsenteret for redskaber som JavaScript og Illustrator. Temaets hovedfokus var rettet mod udviklingen af et website samt et simpelt spil, hvor vi anvendte egendesignede spilelementer. Under designprocessen af spillet benyttede vi os af aktivitetsdiagrammer og Statemachine-diagrammer. Dette tema har forsynet mig med indsigt i, hvordan animation kan gennemføres ved brug af keyframes og JavaScript.
Link til opgave

Illustrator

Alle UI elementer og baggrunde i mit spil er konstrueret ved anvendelse af Illustrator, med undtagelse af et enkelt element i forsidens baggrund. Dette specifikke element udgøres af den store røde planet. Da jeg ikke var tilfreds med min egen udførelse af dette element, gjorde jeg brug af den indlejrede AI, som er indbygget i programmet. Derved opnåede jeg en mere detaljeret realisering af det pågældende element. Udover det fokuserede jeg på at generere dramatiske figurer ved anvendelse af "Pen"-værktøjet i Illustrator.

CSS Keyframes & JavaScript

I dette Tema lærte vi også om CSS animationer eller ”Keyframes” samt grundlæggende principper inden for JavaScript og hvordan de to værktøjer hænger sammen. I eksemplet ved siden af kan man se hvordan. Samspillet mellem CSS Keyframes og JavaScript er konstruktivt, idet Keyframes definerer animationen, og det er ved hjælp af JavaScript, at denne animation anvendes på et specifikt element.
